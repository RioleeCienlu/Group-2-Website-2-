Basic Idea of CSS (Cascading Style Sheets)

CSS (Cascading Style Sheets) is used to style and layout HTML elements, making webpages visually appealing. It controls colors, fonts, spacing, positioning, and responsiveness.

1. How CSS Works

CSS works by selecting HTML elements and applying styles to them. Example:

            <!DOCTYPE html>
            <html>
            <head>
                <style>
                    body {
                        background-color: lightblue;
                    }
                    h1 {
                        color: white;
                        text-align: center;
                    }
                    p {
                        font-size: 18px;
                    }
                </style>
            </head>
            <body>
                <h1>Welcome to My Website</h1>
                <p>This is a styled paragraph.</p>
            </body>
            </html>

2. Ways to Add CSS

There are three main ways to apply CSS to an HTML document:

  1. Inline CSS (Inside Elements) 
    <p style="color: red; font-size: 20px;">This is a red paragraph.</p>
    Good for quick styling but not recommended for large projects.
  2. Internal CSS (Inside <style> in <head>) 
    <style>
    h1 {
        color: blue;
    }
</style>
    Best for single-page styling.
  3. External CSS (Using a Separate .css File) 
    <link rel="stylesheet" href="styles.css">
    styles.css:
h1 {
    color: green;
    font-size: 24px;
}
    Best for large projects as it keeps styles separate from HTML.

3. Basic CSS Selectors

Selectors target elements for styling.

Selector Type Example Description
Element Selector h1 { color: blue; } Targets all <h1> elements
Class Selector .title { font-size: 20px; } Targets elements with class="title"
ID Selector #header { text-align: center; } Targets an element with id="header"
Group Selector h1, p { margin: 10px; } Targets multiple elements at once
Universal Selector * { margin: 0; } Targets all elements

Example:

<h1 class="title">Hello, World!</h1>
<p id="header">This is a paragraph.</p>
.title {
    color: purple;
}
#header {
    font-weight: bold;
}

4. Common CSS Properties

Some commonly used properties:

  • Text & Font Styles
    • color: red; /* Text color */
    • font-size: 16px; /* Text size */
    • font-family: Arial, sans-serif; /* Font type */
    • text-align: center; /* Align text */
    • font-weight: bold; /* Bold text */
  • Box Model (Spacing & Sizing)
    • width: 200px; /* Width of an element */
    • height: 100px; /* Height */
    • padding: 10px; /* Space inside element */
    • margin: 20px; /* Space outside element */
    • border: 2px solid black; /* Border */
  • Background & Colors
    • background-color: lightgray;
    • background-image: url('image.jpg');
  • Positioning & Layout
    • display: flex; /* Flexbox layout */
    • position: absolute; /* Positioning */
    • top: 50px; /* Move element down */
    • left: 100px; /* Move element right */

5. CSS Layout Techniques

Some popular CSS layout techniques include:

  • Flexbox: For flexible layouts.
  • Grid: For advanced layouts.
  • Media Queries: For responsive design (mobile-friendly sites).

Example of a media query:

@media (max-width: 600px) {
    body {
        background-color: yellow;
    }
}
